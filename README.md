# AgroBridge — Agricultural Price Intelligence

## Overview
Analysis of 76,000+ food pricing records from Nigeria's World Food 
Pricing database across 14 states to predict farm produce prices 
and identify price trends.

## Tools Used
- Microsoft Excel
- Power BI

## Key Findings
- Price margins computed across 14 states
- Monthly price trend analysis identified seasonal patterns
- Predictive insights generated for farm produce pricing

## Files in This Repository
- `wfp_food_price_standardized.xlsx` — cleaned dataset and analysed

## Dataset Source
World Food Pricing Nigeria Database (76,000+ records)
